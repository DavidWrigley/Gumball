Welcome

This Webpage has been developed to provide a simple Web Interface that allows a user to register there University of Queensland Student or Staff Card. And provide a full feature set of tools for managment to manage mentioned registered cards

The system could potentialy have many uses including.
  • Tracking Sign-In or Sign-Out times
  • Assignment Submission
  • Card Localisation
  • And many many more
